# URL路径格式使用说明

## 支持的URL格式

### 格式1: 简单关键词格式
```
/keyword1-keyword2-keyword3-keyword4
```
示例：`/web-development-javascript-react-vue`

### 格式2: 路径/文件名.html+关键词格式（您要求的格式）
```
/路径/文件名.html关键词1-关键词2-关键词3-...
```
示例：`/AEDJWOYG/XXTuKkPRGO4EW.htmlsafada-cerda-feet-korean-...`

**重要特点：**
- 文件名（.html）和关键词之间**没有分隔符**
- 关键词用连字符（-）分隔
- 支持任意数量的关键词

### 方法2: 使用URL生成器
1. 打开 `index.html` 页面
2. 在"关键词URL生成器"区域：
   - 选择"路径格式"
   - 输入路径/文件名（例如：`AEDJWOYG/XXTuKkPRGO4EW.html`）
   - 输入关键词（用空格、逗号或换行分隔）
   - 点击"生成URL"
   - 点击"复制URL"或"预览链接"

## 工作原理

1. **Apache重写规则**（.htaccess）
   - 将所有请求重定向到 `index.html`
   - 通过查询参数 `?path=...` 传递原始URL路径

2. **JavaScript解析**
   - 从URL中提取路径部分和关键词部分
   - 识别 `.html` 文件扩展名作为分隔点
   - 将 `.html` 后面的内容按连字符分割为关键词

3. **页面显示**
   - 显示完整URL
   - 显示路径部分（如果有）
   - 显示所有解析出的关键词
   - 显示关键词总数

## 技术细节

### URL解析逻辑
```javascript
// 匹配格式：路径/文件名.html关键词部分
const match = fullPath.match(/^(.+?\.html)([^-].*)?$/i);
```

### 关键词提取
- 文件名和关键词之间没有分隔符
- 第一个关键词紧跟在 `.html` 后面
- 后续关键词用连字符（-）分隔

## 注意事项

1. **URL长度限制**
   - 大多数浏览器和服务器限制URL长度为2048字符
   - 建议关键词总数不超过200个

2. **服务器配置**
   - 需要Apache服务器并启用 `mod_rewrite` 模块
   - 确保 `.htaccess` 文件权限正确

3. **文件扩展名**
   - 目前支持 `.html` 扩展名
   - 可以修改代码支持其他扩展名（.php, .htm等）

## 测试
